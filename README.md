Task Management:- 
A simple web application that allows a **Project Manager (PM)** and **Users** to manage tasks effectively.  
The Project Manager can add, edit, delete, and assign tasks, while users can view assigned tasks and update their status.

---

# Features

## Project Manager (PM)
- Login using username: `pm` and password: `123`
- Add tasks with:
  - Title
  - Description
  - Deadline
  - Assigned user (`user1` / `user2`)
- Edit existing tasks
- Delete tasks with confirmation popup
- View all tasks

## Users (user1 / user2)
- Login using username: `user1` or `user2` and password: `123`
- View only tasks assigned to them
- Update task status (Pending / In Progress / Done)

---

# Optional Feature: Deadline Alert
- If a task **misses its deadline** and is not marked "Done", it will display a warning:
